## Projects

### Park Pulse
A modern Android application built in Kotlin that displays a list of amusement park rides and their current wait times. The app helps visitors plan their park visits by providing real-time ride information and status updates.

**Technologies:**
- Kotlin
- Jetpack Compose
- Navigation Compose
- Hilt (Dependency Injection)
- Retrofit + Gson
- ViewModel + StateFlow
- Material 3
- Clean Architecture

**Key Features:**
- Fetches live ride data from Queue-Times API
- Displays real-time wait times for amusement park rides
- Indicates ride open/closed status
- Reactive state management with StateFlow
- Navigation between list and detail screens
- Modern UI built with Jetpack Compose
- Modular, testable clean architecture
- Material 3 design system
- MVVM architecture pattern

**Repository:** [GitHub - park-pulse](https://github.com/hogandenver05/park-pulse)

- PHP
- MySQL
- JWT Authentication
- Apache Server
- Laravel 12
- PHP 8.2
- Docker & Docker Compose
- Laravel Sanctum
- Eloquent ORM
- Hugo (Documentation)

**Key Features:**
- User registration and authentication (JWT & Laravel Sanctum)
- Full CRUD operations for foods
- Favorite foods management
- RESTful API design
- Docker containerization
- Automated deployment scripts (`run.sh`, `setup.sh`)
- Hugo documentation site
- GitHub Actions for CI/CD (planned)
